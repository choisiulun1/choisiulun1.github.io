## Supervised Machine Learning
- regression : continous numeric value
- classification : discrete label
- examples:
	- k-Nearest Neighbours
	- Linear Regression
	- Logistic Regression
	- SVMs
	- Decision Tree and Random Forests
	- Neural Networks

##  Unsupervised Machine Learning
- data are unlabeled
- Clustering
	- K-Means
	- DBSCAN
	- Hierarchical Cluster Analysis
- Anonaly (異常) and novelty(創意性) detection
	- Isolation Forest
	- One-class SVM

## Semi-supervised Machine Learning
- Some example like Google photos , the machine first categorize the people in the photoes and human manually gives the tag to the machine

## Batch Learning
- The system in Batch Learning can't learn incrementally. It takes expensive resources and a lot of time to learn. So it's done offiline.
- Once it completes learning , it just applies the learning and stop learning anymore

## Online Learning
- Online learning is great for systems that receive data as a continuous flow (e.g., stock prices) and need to adapt to change rapidly or autonomously

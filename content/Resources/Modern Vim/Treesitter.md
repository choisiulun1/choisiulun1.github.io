## What is Treesitter
- Tree-sitter is a parser generator tool and an incremental parsing library. 
- It can build a <mark style="background: #ABF7F7A6;">concrete syntax tree</mark> for a source file and efficiently update the syntax tree as the source file is edited. 


>https://tree-sitter.github.io/tree-sitter/
